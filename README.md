Hospital Management System

A web-based Hospital Management System built using Python and Flask, designed to manage hospital workflows for **Admins, Doctors, and Patients** through role-based dashboards.

Key Features
- Role-based authentication (Admin, Doctor, Patient)
- Admin dashboard to manage doctors, patients, and appointments
- Doctor dashboard to view appointments, update availability, and manage patient history
- Patient dashboard to view departments, doctors, appointments, and medical history
- Appointment scheduling and availability management
- Patient medical records and history tracking

Tech Stack
- Backend: Python, Flask
- Database: SQLite
- Frontend: HTML, CSS
- Tools: Git, GitHub


Note: Features and UI improvements are ongoing.
